# BUILDLoop brand

Palette:
- Primary: #1f4d3a (deep forest green, from existing roadmap)
- Surface: #ffffff / #f6f8f7
- Text: #0d1f17 / #4a5852
- Accent: #c8e6d3 (subtle highlights, evidence badges)

Tone: professional, sober, trustworthy. Building/sustainability without
crunchy-eco visual clichés. Closer to Linear's restraint than Stripe's
bouncy gradients. Users will eventually publish regulated documents
from this tool — the UI must feel like infrastructure, not a startup demo.

Typography: system sans-serif fine for MVP. Use weight and size for
hierarchy, not color.

Avoid: leaf icons, recycling triangles, hand-drawn illustrations,
gradient meshes.
